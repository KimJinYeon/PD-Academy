Introduce Academy.

With PD Academy, you gain confidence in English. Believe and join us!

These days, the English teaching method is floating, the educational policy is changing. There is always a lot of talk about English education. But we have not walked but have walked the road.
I've been thinking a long time about how to learn English naturally as I learned Korean.
As a result, I adopted a learning method using English fairy tales, thinking that watching, listening, and reading in English would be English. All the stories have been animated. In addition, we release new fairy tales every day so that children can see as many stories as possible.
As I quietly walked this path, the world began to acknowledge it. But Little Fox will not be satisfied here. For the day when all the children are learning English happily, we will continue our path one more day.
